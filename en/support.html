1. Product & operator

  • TeLog is a standalone iOS app under the FamilyMedVault brand for Mainland China Tesla owners: lifetime vehicle records, drives, charging, battery insights, PDF export, and share posters.
  • Separate from the main FamilyMedVault web product (family health records, etc.). Vehicle data stays on your phone by default unless you opt into optional cloud features.
  • Not affiliated with Tesla, Inc. Tesla names and marks are used only to describe compatibility.
  • The operator registered a third-party app on Tesla's official China developer portal (developer.tesla.cn) and reads vehicle data via Fleet API only after you authorize on auth.tesla.cn.
  • Operated by an individual developer using familymedvault.com for legal hosting and minimal backend services.
  • Current release supports Mainland China Tesla accounts only.

4. First launch & consent

  1. Privacy & local-first: first launch shows how data stays on-device by default and your rights.
  2. Link Tesla: authorize on auth.tesla.cn with your Mainland China account.
  3. Optional cloud: 24/7 cloud collection and backup require separate consent and usually an active TeLog Plus entitlement.

5. Tesla China Fleet API — authorized data

API access is obtained through the official portal developer.tesla.cn — not unofficial scraping. You authorize on auth.tesla.cn; the app calls fleet-api.prd.cn.vn.cloud.tesla.cn. TeLog never receives your Tesla password.

5.1 Authorization chain

  1. Developer: registered app on developer.tesla.cn with OAuth client_id and Fleet API access.
  2. Owner: you consent on auth.tesla.cn with your Tesla account.
  3. Access: no Fleet API calls without your authorization.

5.2 Data flow

  • Telemetry: app calls Fleet API directly (e.g. vehicle_data) — vehicle payloads are not proxied through our business servers.
  • Token exchange only: OAuth code / refresh via a thin proxy on familymedvault.com (client_secret server-side only).
  • Default storage: encrypted SQLite on your device.
  • Optional upload: only after separate opt-in; copies may be stored in mainland China.

5.3 OAuth scopes (as shown on Tesla's consent screen)

  • openid, offline_access, user_data
  • vehicle_device_data, vehicle_location
  • vehicle_cmds / vehicle_charging_cmds — only when you use vehicle commands

5.4 Data categories

  • Vehicle identity, location & driving, battery & range, charging, vehicle state, climate
  • App annotations: geofence names, reverse-geocoded addresses, manual passport entries (local by default)

Used for stats, battery health, PDF export — not for ads, not sold to third parties.

6. Data storage, export & deletion

  • Default: data stays in on-device encrypted SQLite.
  • Export: Settings → Privacy & data → Export all data (JSON).
  • Delete local: wipe on-device data from the same screen (back up first).
  • Revoke cloud: stop cloud collection and delete remote copies; local data is unaffected.
  • Disconnect Tesla: Advanced settings; also revoke the third-party app in your Tesla account when appropriate.

7. TeLog Plus subscriptions

Auto-renewable subscriptions billed through the Apple App Store. Group name: TeLog Plus.

  • Monthly: com.jchen.Telog.plus.monthly (reference ¥18/mo)
  • Yearly: com.jchen.Telog.plus.yearly (reference ¥198/yr)

7.1 Free trial

14-day free trial for new subscribers (per App Store introductory offer). Cancel anytime in iOS Settings → Subscriptions before trial ends to avoid charges. After expiry, features revert to the Free tier.

7.2 Entitlements

Local — always free: logging & history, local analytics, PDF export, share posters.

Cloud — Plus only: 24/7 cloud collection, nationwide percentile, cloud backup/sync, online share / passport QR, in-app family sharing, remote alerts. Paused when expired; local data is not deleted.

7.3 Apple Family Sharing

Both monthly and yearly plans support Apple Family Sharing per Apple's rules. Distinct from in-app vehicle sharing invites.

7.4 Manage & refunds

  • Manage: iOS Settings → Apple ID → Subscriptions.
  • Refunds: reportaproblem.apple.com — Apple decides; we can confirm subscription status only.

8. Optional cloud features

  • Cloud collection: gap-fill when your phone is offline; separate consent; mainland China storage.
  • Cloud backup / sync: encrypted copies; deleted on server when you turn the feature off.
  • Family location share: separate risk disclosure before enabling.

12. FAQ

Fleet API or sign-in errors?

Use a Mainland China Tesla account on auth.tesla.cn; re-authorize if tokens expire.

Where is data stored?

On-device SQLite by default. Cloud upload requires separate consent.

Delete data?

Settings → Privacy & data, or email us.

Missing drives?

Enable Plus cloud collection or import from TeslaFi / tesla-apiscraper.

Refunds?

Through Apple Report a Problem.

After Plus expires?

Local data remains. Cloud features pause; timeline may show gaps.

14. Compliance (People's Republic of China)

  • We follow applicable PIPL, Cybersecurity Law, and Data Security Law principles: lawful, legitimate, necessary, and good-faith processing.
  • Sensitive data (e.g. precise location, trip tracks) requires separate consent with clear notice.
  • Default processing occurs on your device; uploads, when any, are stored in mainland China as described in the Privacy Policy.
  • Not directed at children under 14; contact us to delete mistaken collection.
